Password Recovery & Reset

Step 1: Request Reset Email

Sends a reset link via email (requires SMTP config). Returns same message regardless of whether email exists (prevents enumeration).

Step 2: Reset Password

Paste the token from the reset email link. All existing sessions are revoked upon reset.

Two-Factor Authentication (2FA)

You must be logged in to use these endpoints. 2FA is optional and user-enabled.

Step 1: Setup 2FA

Generates a TOTP secret, QR code, and recovery codes. Scan the QR code with an authenticator app.

Step 2: Enable 2FA

Enter the 6-digit code from your authenticator app to activate 2FA.
